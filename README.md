# Bible linker Pro

This is an Obsidian plugin for converting Bible texts to links in markdown.

## Features

Converts a Bible text to a JW Library link.

E.g.

ge 1:1

=>

`[Genesis 1:1](jwlibrary:///finder?bible=01001001)`
